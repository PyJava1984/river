Release Notes for com.sun.jini.reggie

Reggie
v2.2.0 Release Notes

    Description

    "Reggie" is the name of the contributed Jini lookup service implementation from Sun Microsystems. The specification for the River lookup service is available in     HTML.

    The specification for Reggie is defined not only by the River Lookup Service Specification, but also by the API documentation (generated by the Javadoc(TM) tool) for Administrable.

    Changes since the v2.1.1 release

    None
