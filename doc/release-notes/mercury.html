Release Notes for Mercury

Mercury
v2.1 Release Notes

    Description

    "Mercury" is the name of the contributed Jini(TM) event mailbox service implementation from Sun Microsystems. The Jini(TM) Event Mailbox Specification is available in     HTML.

    Changes since the v2.0.1 release

    "Pull" Functionality
    The Jini(TM) Event Mailbox Service Specification was augmented with additional classes and interfaces that provide synchronous access to events (see sections EM.5-EM.7). Mercury was also updated to implement this new functionality.
    The following work still needs to be done:
    • Implement the timeout functionality for RemoteEventIterator.next
    • Make the batching implementation configurable. For example, configuring the number of events that get returned to the event mailbox client in a single call. Currently all the available events are sent.
