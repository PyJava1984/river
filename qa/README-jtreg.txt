The jtreg directory contains the jtreg regression and unit tests, each regression test check for the existance of a known bug, each test was set up prior to the bug being fixed to ensure accurate detection.

The jtreg Regression Test Harness can be obtained from http://www.openjdk.org/jtreg/index.html

Once you've installed jtreg, you need to set your JAVA_HOME environment variable to point to your jdk1.5 installation.  Note that jtreg is most compatible with jdk1.5 and should be run with this version for now.
