Provides I/O interfaces and utility classes.

Distributed

The interface {@link org.apache.river.api.io.Distributed} may be implemented by an object that is required to be immutable and thread safe immediately after un-marshalling, Distributed object's do not implement {@link java.io.Serializable}, their fields may be private and they are created by a single reflective invocation of a public constructor, static method or Serializable object's builder method.

DistributedObjectOutputStream and DistributedObjectInputStream

The classes {@link net.jini.io.MarshalOutputStream} and {@link net.jini.io.MarshalInputStream} extend the Distributed object serialization stream classes {@link org.apache.river.api.io.DistributedObjectOutputStream} and {@link org.apache.river.api.io.DistributedObjectInputStream} that implement replacement of {@link org.apache.river.api.io.Distributed} objects with {@link org.apache.river.api.io.SerialReflectionFactory} during marshalling and substitution of SerialReflectionFactory with an object created by SerialReflectionFactoryupon unmarshalling.

SerialReflectionFactory

A {@link org.apache.river.api.io.SerialReflectionFactory} contains Externalized instructions for creating an Object using reflection upon unmarshalling.

DistributePermission

A {@link org.apache.river.api.io.DistributePermission} guards the {@link java.io.Externalizable#writeExternal} method of SerialReflectionFactory, preventing unauthorised marshalling. @since 3.0.0 @version 3.0.0
